# 📉 Macro-Financial Early Warning System (EWS)

This project is an interactive dashboard built using Streamlit to monitor macro-financial risk using Credit-to-GDP Gap.

## 🚀 Features
- Credit-to-GDP visualization
- HP Filter trend analysis
- Early Warning Signal (EWS)
- Scenario simulation
- ASEAN comparison (peer analysis)

## 📊 Data Source
- World Bank API (Credit to Private Sector as % of GDP)

## 🧠 Methodology
- Hodrick-Prescott (HP) Filter
- Credit Gap calculation
- Threshold-based Early Warning Signal

## 🛠️ Tech Stack
- Python
- Streamlit
- Pandas
- Statsmodels
- Matplotlib

## ▶️ How to Run
```bash
pip install -r requirements.txt
streamlit run app.py
